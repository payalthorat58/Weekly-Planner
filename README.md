# Weekly-Planner
Merged some features of my top two productivity apps- Notion and Focus To-Do through vibe coding.
1. You can create weekly plans, monthly plans in it.
2. You can also add pomodoro sessions to specific task.
3. You can also add your own background image for pomodoro sessions by pasting it's URL.
4. You can track the number of hours of pomodoro sessions.
5. You can edit, delete your task later.
6. You can get daily, weekly, monthly and yearly statistics as well.
7. Light/Dark both modes available.
8. You can select the week start day - Monday/Sunday.
9. You can select a particular task as permanent/repetitive, the weekly repeated tasks doesn't need to be added differently.
10. Added reminders, you can select on which day and what time you need to be reminded about certain task.
11. Clicking on cross mark will mark the task incomplete for that day.
